TMX(Translation Memory eXchange) tools for ruby

## Installation

Add this line to your application's Gemfile:

    gem 'konjak'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install konjak

## Usage

    $ konjak translate src target file.tmx file.txt
